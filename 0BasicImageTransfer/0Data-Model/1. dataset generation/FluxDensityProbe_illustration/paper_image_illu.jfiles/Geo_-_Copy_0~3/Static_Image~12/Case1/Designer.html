Automatic CG acceleration factor determination function is not supported under the following conditions when [Shared Memory Multiprocessing (SMP)] is selected in the [Parallel Computing Settings]. Please open the [Study Properties] dialog, move on to the [Linear Solver] setting, then clear the [Set Automatically] checkbox for the [Acceleration Coefficient] and set the fixed value. Higher value is preferred for the acceleration coefficient such as 1.2 or 1.4.
